config module
=============

Configurations for fonts, colors, titles, imported files for GUI.

:code:`TITLE` (:code:`str`): title string for StartPage

:code:`APP_LOGO`: path for app icon file

:code:`TITLE_FONT` (:code:`tuple`): font configuration for :code:`TITLE`

:code:`SUBTITLE_FONT` (:code:`tuple`): font configuration for titles in other pages

:code:`LARGE_FONT`, :code:`NORMAL_FONT` (:code:`tuple`): variations of font configuration

:code:`*FG*` (:code:`str`): foreground colors in hex code

:code:`*BG*` (:code:`str`): background colors in hex code
